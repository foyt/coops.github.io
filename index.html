CoOps

CoOps is a open generic protocol for online collaboration

So how it's different from all the other cool collaboration thingies in the internet?

It's true that for example Etherpad is quite well-known open-source tool for collaborating documents and everyone knows that you can edit simultaneously pretty much anything with Google Drive

So what makes CoOps any different from those?

Well, first of all it's a protocol not an application. That means that developers can build any kind of collaboration applications on top of it

CoOps is designed to be extremely easy to implement, in both server and client side and because it's based on text patches you can create an editor for pretty much anything you wish

It's also free, open and you can participate into it's development if you wish

Does that mean that i can not edit a document with my friends using CoOps

Yes and no. Yes, you can not edit anything with a protocol but you can use an applications build on top of it. You can for example try it on the CoOps demo

Btw. If you have created an application and wish it to be listed here, please drop me an email at development(at)foyt.fi or add an issue into https://github.com/foyt/coops.github.io/issues and i will gladly add it here

So can i create an application using the CoOps?

Now we are talking! First of all you should check out specifications from https://github.com/foyt/coops-spec

After reading that you should have pretty good picture how the protocol works.

So i need a Server

Yes, That right! You need a server. Currently there's no general fits-for-all server available but don't worry, it's not even nearly as hard as you think to roll your own.

It's true that creating a collaboration server could be a pain but it's not because we already have two separate example implementations that will make server implementation much easier. Here you go:

  • Quick start for CoOps Java server
  • CoOps Demo (NodeJs)

...and i need a client

Awesome! You are getting hang of it. Depending on what the client should be able to do it can be either quite tricky or very easy (keep in mind that CoOps has been developed simplicity in mind)

You can of course roll your own client or use one of already existing projects:

  • CoOps add-on for CKEditor (Rich text editor)
  • CoIllusionist image editor (as part of CoOps demo)
